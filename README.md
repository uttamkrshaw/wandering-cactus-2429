## Run Locally

Clone the project

```bash
  git clone https://github.com/uttamkrshaw/wandering-cactus-2429
```

Go to the project directory

```bash
  cd sasta_bazar
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
